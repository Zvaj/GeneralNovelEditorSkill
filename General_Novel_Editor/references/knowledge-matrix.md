# Knowledge Matrix

Who knows what, and when they learned it. This is the single most common place plot holes hide — a character acts on information they shouldn't have, or fails to act on something they do know.

Update this whenever a character learns, forgets, or is told something important.

---

## How to use this file

Each row is a **piece of information**. Each column is a **character**. The cell says when (chapter/scene) that character learned it, or "—" if they don't know. "Suspects" is distinct from "knows." "Lied to about it" is distinct from "doesn't know."

Abbreviations:
- **K** = knows (with source chapter)
- **S** = suspects (with source)
- **L** = been lied to / has wrong info (with source)
- **—** = doesn't know
- **F** = forgot / memory altered (with source)

## Active Deceptions
